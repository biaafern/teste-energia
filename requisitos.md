# Requisitos da Área de Energia

* **RF-ENER-01**
A fonte de energia deve ser altamente compacta e apresentar uma excelente relação peso/potência, garantindo que o chassi permaneça leve.

* **RF-ENER-02**
O sistema de alimentação deve utilizar baterias adequadas para suprir a demanda por um período contínuo de 20 a 30 minutos em regime de testes.

* **RF-ENER-03**
O conjunto de baterias deve garantir energia suficiente para os componentes e sustentar os picos elétricos do arranque dos motores.

* **RF-ENER-04**
O projeto deve integrar um BMS como interface de proteção contra descarga profunda, sobrecarga e curtos-circuitos.
